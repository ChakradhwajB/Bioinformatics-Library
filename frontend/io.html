Educational Guide: FASTA File Ingestion & JSON Exporters

In bioinformatics, storing and sharing genetic sequences is standardly performed using text-based file formats. The most common of these is the FASTA format.

1. FASTA File Structure

A FASTA file represents nucleotide sequences or peptide sequences using single-letter codes. A sequence in FASTA format begins with a single-line description/header, followed by lines of sequence data. The description line is distinguished from the sequence data by a greater-than symbol (>) at the beginning of the line: 

>MCHU - Mycoplasma chromosome fragment
ASEUIAHSEUIHASEUIHASEUIAHSEUIAHSE

2. Linear Parsing Mechanics

The parser scans the file line-by-line in $\mathcal{O}(n)$ time complexity. It ignores whitespaces and joins lines belonging to the same identifier header, creating a clean key-value mapping of records.
