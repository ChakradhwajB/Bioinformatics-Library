Educational Guide: Dot Plot Matrices

1. Motivation: Why care?

When comparing two long sequences, single mathematical scores (like an alignment score) don't tell the whole story. A Dot Plot provides an immediate visual footprint of macro-structural relationships—like large insertions, tandem repeats, or inversions—that alignment algorithms might miss or mathematically compress.

2. Concept & Complexity

The matrix compares all characters of sequence $S_1$ (rows) against sequence $S_2$ (columns). In its simplest form, a dot is plotted at cell $(i, j)$ if $S_1[i] = S_2[j]$. To filter background noise, we use a sliding window of size $w$ and a match threshold $t$.

  • Time Complexity: $\mathcal{O}(n \cdot m \cdot w)$ where $n = |S_1|$, $m = |S_2|$, and $w$ is the sliding window size.
  • Space Complexity: $\mathcal{O}(n \cdot m)$ to store the 2D Boolean matrix.

3. Worked Example

Compare AAT (vertical) vs GAT (horizontal) with window size 1, threshold 1.

G A T A . * . A . * . T . . *

Notice the diagonal streak forming on the bottom right, indicating the matching suffix AT.

4. Python Code

Here is how to generate a comparative dot plot matrix in Python using sliding window matches:

def generate_dot_plot(seq1, seq2, window_size, threshold):
    n, m = len(seq1), len(seq2)
    matrix = [[False for _ in range(m)] for _ in range(n)]
    
    for i in range(n - window_size + 1):
        for j in range(m - window_size + 1):
            matches = 0
            for w in range(window_size):
                if seq1[i + w] == seq2[j + w]:
                    matches += 1
            if matches >= threshold:
                matrix[i][j] = True
                
    return matrix

Core

Write a function that analyzes a boolean dot plot matrix to find and return the length of the longest continuous diagonal line.

Extension
