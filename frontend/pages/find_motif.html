Educational Guide: Motif Finding & Spotlight Vectors

1. Motivation: Why care?

In bioinformatics, locating specific sequence motifs within larger genomic sequences is a crucial task for understanding transcriptional regulation, gene expression, and DNA-protein binding interactions. For example, RNA polymerase binds to specific promoter motifs to initiate transcription.

2. Concept & Complexity

A sequence motif is a short, recurring pattern of nucleotides or amino acids that has biological significance. The motif finder performs a linear search over a genome sequence $S$ of length $n$ to identify all occurrences of a target motif sequence $M$ of length $m$.

  • Time Complexity: $\mathcal{O}(n \cdot m)$ worst-case using a naive sliding window comparison.
  • Space Complexity: $\mathcal{O}(k)$ where $k$ is the number of motif matches found (to store their index positions).

3. Worked Example

Consider searching for the motif "ATA" inside the sequence "GATATATA". We slide a window of size 3.

Index 0: GAT != ATA Index 1: ATA == ATA (Match at 1) Index 2: TAT != ATA Index 3: ATA == ATA (Match at 3) Index 4: TAT != ATA Index 5: ATA == ATA (Match at 5)

Notice that biologically, motifs can overlap (indices 1, 3, and 5).

4. Python Code

Here is how to search for motifs and map their overlapping positions in Python:

def find_motifs(sequence, motif):
    match_offsets = []
    n, m = len(sequence), len(motif)
    
    # Slide a window of size m across the sequence
    for i in range(n - m + 1):
        if sequence[i : i + m] == motif:
            match_offsets.append(i)
            
    return match_offsets

5. Practice Problems

Warm-up

Manually find all starting 0-indices of the motif "CGC" in the sequence "ACGCGATCGCCG".

Core

Write a function find_motif_distance(seq, motif) that returns the minimum distance between any two consecutive occurrences of the motif. Return -1 if the motif appears less than twice.

Extension

The naive string matcher takes $\mathcal{O}(n \cdot m)$ time. Research and implement the Knuth-Morris-Pratt (KMP) algorithm to achieve $\mathcal{O}(n + m)$ linear time complexity.
