Educational Guide: Suffix Arrays

1. Motivation: Why care?

Suffix array search takes $\mathcal{O}(|P| \log N)$ time, where $|P|$ is the pattern length and $N$ is text length. This is significantly faster than sequential substring scanning on extremely large genomes.

2. Concept & Complexity

A Suffix Array is a lexicographically sorted array of all suffixes of a text sequence. Because all suffixes are sorted, any matching occurrences of a query pattern will appear as prefixes of a contiguous block of suffixes in the suffix array. We locate this matching range using binary search.

  • Time Complexity: $\mathcal{O}(|P| \log N)$ to binary search for a pattern of length $|P|$.
  • Space Complexity: $\mathcal{O}(N)$ to store the array of suffix offsets.

3. Worked Example

Text BANA$:

Suffixes: [0]BANA$, [1]ANA$, [2]NA$, [3]A$, [4]$ Sorted: [4]$, [3]A$, [1]ANA$, [0]BANA$, [2]NA$ Suffix Array: [4, 3, 1, 0, 2]

4. Python Code

Here is how to construct a Suffix Array and perform binary search queries in Python, compared against a naive scanning approach:

Naive Substring Scanning:

def naive_substring_search(text, pattern):
    offsets = []
    n, m = len(text), len(pattern)
    for i in range(n - m + 1):
        if text[i : i + m] == pattern:
            offsets.append(i)
    return offsets

Suffix Array Construction & Binary Search:

def build_suffix_array(text):
    text_with_sentinel = text + "$"
    # Generate all suffixes with starting offsets
    suffixes = [(text_with_sentinel[i:], i) for i in range(len(text_with_sentinel))]
    # Sort lexicographically by suffix string
    suffixes.sort()
    # Extract only the suffix offset indices
    return [item[1] for item in suffixes]

def binary_search_suffix_array(text, suffix_array, pattern):
    text_with_sentinel = text + "$"
    n = len(text_with_sentinel)
    
    # 1. Lower Bound Binary Search
    low, high = 0, n - 1
    left_bound = n
    while low <= high:
        mid = (low + high) // 2
        suffix = text_with_sentinel[suffix_array[mid]:]
        if suffix.startswith(pattern) or suffix >= pattern:
            left_bound = mid
            high = mid - 1
        else:
            low = mid + 1
            
    # 2. Upper Bound Binary Search
    low, high = 0, n - 1
    right_bound = -1
    while low <= high:
        mid = (low + high) // 2
        suffix = text_with_sentinel[suffix_array[mid]:]
        if suffix.startswith(pattern):
            right_bound = mid
            low = mid + 1
        elif suffix < pattern:
            low = mid + 1
        else:
            high = mid - 1
            
    # Return matched start offsets
    if left_bound <= right_bound:
        return sorted([suffix_array[i] for i in range(left_bound, right_bound + 1)])
    return []
