Educational Guide: Global Sequence Alignment

1. Motivation: Why care?

Without sequence alignment, comparing genetic material between organisms to find evolutionary relationships or functional similarities is impossible. Needleman-Wunsch is the fundamental dynamic programming algorithm that guarantees finding the mathematically optimal global alignment between two sequences from end-to-end. It forms the basis of homology search and evolutionary distance calculation.

2. Concept & Complexity

Let $S_1$ and $S_2$ be sequences of lengths $n$ and $m$. We construct an $(n+1) \times (m+1)$ score grid $E$. The recurrence relation is:

$$E_{i, j} = \max \begin{cases} E_{i-1, j-1} + S(S_1[i-1], S_2[j-1]) & \text{(Match / Mismatch)} \\ E_{i-1, j} + \text{gap} & \text{(Deletion)} \\ E_{i, j-1} + \text{gap} & \text{(Insertion)} \end{cases}$$

where the substitution score is $S(a,b) = \text{match}$ if $a = b$, and $\text{mismatch}$ if $a \neq b$.

  • Time Complexity: $\mathcal{O}(n \cdot m)$ since every cell in the grid requires constant $\mathcal{O}(1)$ calculations.
  • Space Complexity: $\mathcal{O}(n \cdot m)$ to store the entire score matrix for traceback.

3. Worked Example

Let's align sequence $S_1 = \text{"GC"}$ (vertical) and $S_2 = \text{"GA"}$ (horizontal) with scoring: Match = 1, Mismatch = -1, Gap = -1.

- G A - 0 -1 -2 G -1 1 0 C -2 0 0

Traceback starting at bottom-right cell score 0 yields the optimal alignment:
S1: G C
S2: G A (Score: 0)

4. Python Code

Here is the implementation of the Needleman-Wunsch global alignment algorithm in Python:

def needleman_wunsch(seq1, seq2, match=1, mismatch=-1, gap=-1):
    n, m = len(seq1), len(seq2)
    score = [[0 for _ in range(m + 1)] for _ in range(n + 1)]
    
    # Base cases
    for i in range(n + 1):
        score[i][0] = i * gap
    for j in range(m + 1):
        score[0][j] = j * gap
        
    # Fill the matrix
    for i in range(1, n + 1):
        for j in range(1, m + 1):
            sub_score = match if seq1[i - 1] == seq2[j - 1] else mismatch
            score[i][j] = max(
                score[i - 1][j - 1] + sub_score,
                score[i - 1][j] + gap,
                score[i][j - 1] + gap
            )
            
    # Traceback
    align1, align2 = [], []
    i, j = n, m
    while i > 0 or j > 0:
        if i > 0 and j > 0 and score[i][j] == score[i - 1][j - 1] + (match if seq1[i - 1] == seq2[j - 1] else mismatch):
            align1.append(seq1[i - 1])
            align2.append(seq2[j - 1])
            i -= 1
            j -= 1
        elif i > 0 and score[i][j] == score[i - 1][j] + gap:
            align1.append(seq1[i - 1])
            align2.append("-")
            i -= 1
        else:
            align1.append("-")
            align2.append(seq2[j - 1])
            j -= 1
            
    return "".join(align1[::-1]), "".join(align2[::-1])
