Educational Guide: Central Dogma Transformations

1. Motivation: Why care?

The Central Dogma of molecular biology explains how genetic information flows from DNA to RNA to proteins. Understanding how to computationally replicate this process allows us to translate a raw genomic sequence into the amino acid string that builds actual functioning proteins. This is the foundational step for understanding gene expression.

2. Concept & Complexity

The transformation involves three operations:

  • Reverse Complement: Generating the opposing DNA strand (A ↔ T, C ↔ G) and reversing its direction (5' → 3').
  • Transcription: Converting DNA to RNA by substituting Thymine (T) with Uracil (U).
  • Translation: Mapping triplets of RNA nucleotides (codons) to amino acids using a standard codon table. Halts upon a "Stop" codon.
  • Time Complexity: O(N) for all transformations, where N is the sequence length.
  • Space Complexity: O(N) to store the resulting strings.

3. Worked Example

Start with the DNA sequence TACGGC.

1. Transcribe (T → U): UACGGC 2. Reverse Complement (A↔T, C↔G, reversed): GCCGTA 3. Translate "UACGGC" (Codons: UAC, GGC): Y G (Tyrosine, Glycine)

4. Python Code

Here are the implementations for basic sequence manipulations:

def transcribe_dna(dna):
    return dna.replace("T", "U")

def reverse_complement(dna):
    complements = {"A": "T", "T": "A", "C": "G", "G": "C"}
    comp_strand = [complements.get(char, char) for char in dna]
    return "".join(comp_strand[::-1])

CODON_TABLE = { "AUG": "M", "UUU": "F", "UAA": "Stop" # (abridged) }

def translate_rna(rna):
    protein = []
    for i in range(0, len(rna) - 2, 3):
        codon = rna[i : i + 3]
        amino_acid = CODON_TABLE.get(codon, "?")
        if amino_acid == "Stop":
            break
        protein.append(amino_acid)
    return "".join(protein)

5. Practice Problems

Warm-up

Write the reverse complement of ATGCATGC by hand.

Core

Write a function that counts how many times the amino acid "Methionine" (M) appears in a given RNA strand translation.

Extension

Implement an Open Reading Frame (ORF) finder that scans all 3 possible reading frames to find the longest continuous translated protein chain before hitting a Stop codon.
