Educational Guide: Trie (Prefix Trees)

1. Motivation: Why care?

For genetic sequences, Tries offer a powerful way to match multiple motifs concurrently. Instead of scanning the genome independently $M$ times for $M$ patterns, we build a single Trie and match all patterns in one pass.

2. Concept & Complexity

A Trie (from "retrieval") is a tree-like data structure used to store a set of strings, where each path from the root to a node represents a common prefix shared by multiple strings.

  • Time Complexity: $\mathcal{O}(N \cdot L)$ where $N$ is sequence length and $L$ is max pattern length.
  • Space Complexity: $\mathcal{O}(P \cdot |\Sigma|)$ where $P$ is the total bases in all patterns and $|\Sigma|=4$ for DNA.

3. Worked Example

Insert AT and AG into a Trie.

Root -> A -> T (end of word "AT") -> G (end of word "AG")

4. Python Code

Here is how to construct a prefix tree and search it in Python, compared against a naive sliding-window approach:

Naive Multi-Pattern Search (Sliding Window):

def naive_search(sequence, patterns):
    results = {}
    for pattern in patterns:
        results[pattern] = []
        n, m = len(sequence), len(pattern)
        for i in range(n - m + 1):
            if sequence[i : i + m] == pattern:
                results[pattern].append(i)
    return results

Efficient Trie Multi-Pattern Search:

class TrieNode:
    def __init__(self):
        self.children = {}
        self.is_end_of_word = False

class Trie:
    def __init__(self):
        self.root = TrieNode()

    def insert(self, word):
        node = self.root
        for char in word:
            if char not in node.children:
                node.children[char] = TrieNode()
            node = node.children[char]
        node.is_end_of_word = True

def find_patterns_trie(sequence, patterns):
    trie = Trie()
    for p in patterns:
        trie.insert(p)
        
    results = {p: [] for p in patterns}
    n = len(sequence)
    for i in range(n):
        node = trie.root
        for j in range(i, n):
            char = sequence[j]
            if char not in node.children:
                break
            node = node.children[char]
            if node.is_end_of_word:
                matched_pattern = sequence[i : j + 1]
                if matched_pattern in results:
                    results[matched_pattern].append(i)
    return results

Draw a Trie containing the words "CAT", "CAR", and "CART".

Core

Modify the Trie insertion code so that each node also stores the number of times it has been visited. How can you use this to find the most frequent prefix?

Extension

Implement the Aho-Corasick algorithm, which extends the Trie with failure links to achieve $\mathcal{O}(N + P + k)$ time matching.
