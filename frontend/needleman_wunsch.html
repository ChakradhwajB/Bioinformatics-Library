Educational Guide: Global Sequence Alignment

The Needleman-Wunsch algorithm is a dynamic programming method designed to compute the optimal global alignment between two sequences. Global alignment attempts to match both sequences from end-to-end, which is ideal for closely related sequences of similar length.

1. Mathematical Model & Transition Rule

Let $S_1$ and $S_2$ be sequences of lengths $n$ and $m$. We construct an $(n+1) \times (m+1)$ score grid $E$, where cell $E_{i,j}$ represents the maximum alignment score for prefixes $S_1[0 \dots i-1]$ and $S_2[0 \dots j-1]$. The recurrence relation is formulated as:

$$E_{i, j} = \max \begin{cases} E_{i-1, j-1} + S(S_1[i-1], S_2[j-1]) & \text{(Match / Mismatch)} \\ E_{i-1, j} + \text{gap} & \text{(Deletion)} \\ E_{i, j-1} + \text{gap} & \text{(Insertion)} \end{cases}$$

where the substitution score is $S(a,b) = \text{match}$ if $a = b$, and $\text{mismatch}$ if $a \neq b$.

2. Step-by-Step Execution Workflow

  • Initialization: The first row and column are filled with cumulative gap penalties: $E_{i, 0} = i \times \text{gap}$ and $E_{0, j} = j \times \text{gap}$. This represents aligning prefixes against empty sequences.
  • Matrix Fill: The grid is filled row-by-row or column-by-column. Every cell value relies solely on its diagonal, top, and left neighbors.
  • Traceback: To reconstruct the actual alignment, we start at the bottom-right corner $E_{n,m}$ and follow the score transitions backwards. A diagonal step represents aligning characters; a vertical step represents a deletion (gap in $S_2$); a horizontal step represents an insertion (gap in $S_1$).

3. Algorithmic Complexity Analysis

Both Time and Space complexities are quadratic: $\mathcal{O}(n \cdot m)$. This is because every cell in the grid requires constant $\mathcal{O}(1)$ calculations, and we must store the entire matrix in memory to trace back the optimal path.
