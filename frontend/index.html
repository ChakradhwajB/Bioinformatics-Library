Step 04 / Dynamic Programming Alignments

Module 4: Global & Local Sequence Alignment

Apply dynamic programming to optimally align two biological sequences. Use Needleman-Wunsch for end-to-end global alignment of homologous genes, or Smith-Waterman for local alignment of conserved domains within larger sequences.

Alignment Hard

Needleman-Wunsch Global

Perform end-to-end global alignment. Includes animated traceback path reconstruction to visualize the optimal sequence match across the matrix.

$$D_{i,j} = \max(D_{i-1,j-1} + S, D_{i-1,j} + g, D_{i,j-1} + g)$$
Launch Module →
Alignment Hard

Smith-Waterman Local

Identify the most similar local sub-regions. Includes zero-score thresholds and terminated traceback logic to isolate high-scoring domains.

$$D_{i,j} = \max(0, D_{i-1,j-1} + S, D_{i-1,j} + g, D_{i,j-1} + g)$$
